# UX Specification

## UX Goals
- Complete a mission in under 60 seconds.
- Minimize typing.
- Mobile-first.
- Offline-first.

## Primary Flow
Login → Mission Feed → Mission → Capture → Review → Submit

## Screens
- Splash
- Login
- Home
- Mission
- Capture
- Review
- Profile

## UX Rules
- One primary action per screen.
- Camera opens in one tap.
- GPS captured automatically.
- AI suggestions are editable.
