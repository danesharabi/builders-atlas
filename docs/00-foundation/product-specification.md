# Product Specification

## Purpose

Define the scope, objectives and functional boundaries of Builders
Atlas.

## Product Goal

Build a community-powered operating system for collecting, validating
and evolving local knowledge.

## Primary Users

-   Contributors
-   Stewards
-   Administrators

## Core Modules

-   Knowledge Capture
-   Missions
-   Verification
-   Search
-   Knowledge Graph
-   AI Assistant
-   Analytics

## MVP

Capture observations, validate them, organize them into entities and
relationships.
